# YAMA Osmanisch-Lektüre 📜

**Osmanlıca okuma çalışmaları için web tabanlı ders/okuma materyali**
*Webbasierte Lektürematerialien für das Studium des Osmanisch-Türkischen*

Osmanlı Türkçesi metinlerini okumayı öğrenenler için hazırlanmış, tarayıcıda
çalışan bir okuma platformu. Kurulum ve sunucu gerektirmez.

## İçerik

- `index.html` — giriş sayfası
- `osmanisch-lektuere.html` — okuma modülü

## Kullanım

`index.html` dosyasını tarayıcıda açmanız yeterlidir.
GitHub Pages ile yayınlamak için: Settings → Pages → Branch: `main` seçin.

## Hedef kitle

Turkoloji öğrencileri, Osmanlıca öğrenenler ve tarihî metinlerle çalışan
araştırmacılar. [YAMA-Turkologie-Guide](https://github.com/ergunozsoy/YAMA-Turkologie-Guide)
ile birlikte kullanılabilir.
